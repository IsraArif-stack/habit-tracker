# 🔥 Habit Tracker with Streak Heatmap

A modern habit tracking web application built with **Next.js** that helps users build consistency by tracking daily habits, maintaining streaks, and visualizing progress through a GitHub-style contribution heatmap.

The application provides motivation and feedback through streak tracking, progress analytics, achievement badges, and activity visualization.

---

# 🚀 Features

## ✅ Habit Management

- Add new habits
- Delete habits
- Prevent duplicate habits
- Complete habits daily
- Store data using browser Local Storage

---

## 🔥 Streak Tracking

- Current streak calculation
- Longest streak tracking
- Daily habit completion tracking
- Motivation through consistency monitoring

---

## 📅 GitHub Style Heatmap

- 365 days activity visualization
- Contribution-style heatmap grid
- Shows completed habit days visually
- Helps users analyze consistency

---

## 📊 Dashboard Analytics

- Total habits count
- Total completed days
- Today's completed habits
- Weekly activity chart
- Progress visualization

---

## 🏆 Achievement Badge System

Users receive badges according to their streak progress:

🥉 Bronze Badge  
- 3 Days Streak

🥈 Silver Badge  
- 7 Days Streak

🥇 Gold Badge  
- 30 Days Streak

👑 Legend Badge  
- 100 Days Streak

---

## 🌙 Dark Mode

- Light and dark theme support
- Theme switching button
- User preference saved automatically

---

## 🔍 Search Feature

- Search habits instantly
- Filter habits quickly
- Better user experience

---

## 🔔 Toast Notifications

Modern notifications for:

# 📦 Installation & Setup

Follow these steps to run the project locally.

## 1. Clone Repository

```bash
git clone https://github.com/your-username/habit-tracker.git
```

---

## 2. Navigate to Project Folder

```bash
cd habit-tracker
```

---

## 3. Install Dependencies

```bash
npm install
```

---

## 4. Start Development Server

```bash
npm run dev
```

Open your browser:

```
http://localhost:3000
```
